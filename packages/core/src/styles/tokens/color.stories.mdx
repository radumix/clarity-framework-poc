# Color

Clarity provides several color options when building your applications. The global
palette provides a large selection of Clarity themed colors. The alias tokens provides
a set of semantic purpose defined tokens for different UI interactions and states.

## Status

Status colors provide a palette of colors for conveying status information. Our
status colors consist of info (default), warning, danger, and success.
